## 常见问题

### 国内可以直接开通 Grok Super 吗？

可以尝试官方订阅，但国内用户经常遇到海外支付、地区限制和付款失败。如果没有合适的支付方式，可以使用微信卡密充值。

### Grok 充值需要提供账号密码吗？

不建议提供账号密码。更稳妥的做法是自己登录账号，获取账号 ID，并在充值确认页核对账号信息。

### 账号 ID 在哪里获取？

先登录目标 Grok / X 账号，再访问 `https://grok.com/api/auth/session`。如果使用充值页面里的超链接，也要先确认浏览器已经登录正确账号。

### 账号 ID 复制错了怎么办？

如果还没提交，重新登录目标账号后再获取一次。如果已经提交，不要重复下单，保留订单号和错误信息联系售后。

### Grok Super 和 SuperGrok 是一个东西吗？

用户搜索时经常混用这两个名称，但具体套餐名和权益可能随官方页面调整。下单前以购买页面和 Grok / X 页面展示为准。

### 充值后多久生效？

通常需要等待一段处理时间，具体取决于账号状态和平台处理速度。短时间没显示时，可以刷新、重新登录，或联系售后核对。
