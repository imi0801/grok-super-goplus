## 常见问题

### 国内可以直接开通 Grok Super 吗？

可以尝试官方订阅，但国内用户可能遇到支付方式、地区和订阅入口问题。没有合适支付方式时，可以使用卡密充值。

### Grok Super 和 SuperGrok 是一个东西吗？

用户搜索时经常混用这两个名称，具体套餐名和权益可能随官方页面变化。下单前以购买页面和 Grok / X 页面展示为准。

### Grok 充值需要账号密码吗？

不建议提供账号密码。更稳妥的方式是自己登录账号，获取账号 ID，并在确认页核对信息。

### 账号 ID 在哪里获取？

先登录目标 Grok / X 账号，再访问 `https://grok.com/api/auth/session`。如果使用充值页面里的获取链接，也要先确认浏览器登录正确账号。

### 账号 ID 可以填邮箱吗？

不要直接填邮箱。充值页面需要的是账号 ID，应按页面提示或 session 页面返回的信息填写。

### 账号 ID 复制错了怎么办？

如果还没提交，重新登录目标账号后再获取一次。如果已经提交，不要重复下单，保留订单记录联系售后。

### 账号 ID 页面打不开怎么办？

先确认已经登录 Grok / X 账号，再刷新或换浏览器访问。仍无法打开时，可以尝试充值页面里的获取链接。

### 充值后多久生效？

通常需要等待一段处理时间。短时间没显示时，可以刷新或重新登录；长时间无变化，再联系售后核对。

### 充值前最容易出错的地方是什么？

最容易出错的是浏览器登录了错误账号，或者复制了不完整的 ID。提交前要看确认页，不要只看自己复制的内容。

### Grok、ChatGPT、Claude 怎么选？

Grok 更适合实时资讯和 X 内容分析；ChatGPT 更适合综合办公、写作、代码和多模态任务；Claude 更适合长文档和复杂资料整理。
